## multiply 方法
::: tip 常规用法
multiply(multiplier: number, multiplicand: number): number

乘法运算，解决精度问题。
:::


### multiply 参数列表
| 参数          | 说明                 | 类型     | 默认值   |
| :----------- |:-----------------| :--------| :--------|
| multiplier         | 乘数     | number | -|
| multiplicand         | 被乘数     | number | - |

#### 例子
```js
multiply(0.2, 0.1)
// --> 0.02

multiply(6.66, 3.7)
// --> 24.642

```
