## intersection 方法
::: tip 常规用法
intersection([arrays]): array

创建唯一值的数组，这个数组包含所有给定数组都包含的元素（注：可以理解为给定数组的交集）
:::

### 参数列表
| 参数          | 说明          | 类型         | 默认值   |
| :------------ |:------------ | :--------    | :--------|
| arrays      | 要检查的数组。   | (...Array) | -      |

### 返回值 
array: 返回一个包含所有传入数组交集元素的新数组。

### 例子
```js
intersection([2, 1], [4, 2], [1, 2]);
// => [2]
```
