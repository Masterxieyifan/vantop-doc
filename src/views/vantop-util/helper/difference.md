## difference 方法
::: tip 常规用法
_.difference(array, [values]): array

创建一个具有唯一array值的数组，每个值不包含在其他给定的数组中。（注：即创建一个新数组，这个数组中的值，为第一个数字（array 参数）排除了给定数组中的值。）结果值的顺序是由第一个数组中的顺序确定。
:::

### 参数列表
| 参数          | 说明          | 类型         | 默认值   |
| :------------ |:------------ | :--------    | :--------|
| array      | 要检查的数组。   | Array | -      |
| [values] (...Array)      | 排除的值  | (...Array) | - |

### 返回值 
array: 返回一个过滤值后的新数组。

### 例子
```js

difference([3, 2, 1], [4, 2]);
// => [3, 1]

```
