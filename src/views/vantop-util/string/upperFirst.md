## upperFirst 方法
::: tip 常规用法
upperFirst(str: string): string 

返回首字母大写的字符串
:::


### upperFirst 参数列表
| 参数          | 说明                 | 类型     | 默认值   |
| :----------- |:-----------------| :--------| :--------|
| str         | 需要转换的字符串     | string       | -      |

#### 例子

```js
upperFirst('nextop'); 
// --> 'Nextop'
```
