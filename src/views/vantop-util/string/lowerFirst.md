## lowerFirst 方法
::: tip 常规用法
lowerFirst(str: string): string 

返回首字母小写的字符串
:::


### lowerFirst 参数列表
| 参数          | 说明                 | 类型     | 默认值   |
| :----------- |:-----------------| :--------| :--------|
| str         | 需要转换的字符串     | string       | -      |

#### 例子

```js
lowerFirst('Nextop'); 
// --> 'nextop'
```
