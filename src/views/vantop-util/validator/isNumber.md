## isNumber 方法
::: tip 常规用法
isNumber(value): boolean 

若value为数字类型，则返回true，否则返回false。
:::


### isNumber 参数列表
| 参数          | 说明                 | 类型     | 默认值   |
| :----------- |:-----------------| :--------| :--------|
| value         | 需要验证的值     | any       | -      |

#### 例子

```js
isNumber(123)
// --> true
```
