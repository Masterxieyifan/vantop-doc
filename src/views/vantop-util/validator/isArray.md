## isArray 方法
::: tip 常规用法
isArray(value): boolean 

若value为数组，则返回true，否则返回false。
:::


### isArray 参数列表
| 参数          | 说明                 | 类型     | 默认值   |
| :----------- |:-----------------| :--------| :--------|
| value         | 需要验证的值     | any       | -      |

#### 例子

```js
isArray([1,2,3,4])
// --> true
```
