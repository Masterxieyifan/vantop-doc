## isFunction 方法
::: tip 常规用法
isFunction(value): boolean 

若value为方法，则返回true，否则返回false。
:::


### isFunction 参数列表
| 参数          | 说明                 | 类型     | 默认值   |
| :----------- |:-----------------| :--------| :--------|
| value         | 需要验证的值     | any       | -      |

#### 例子

```js
isFunction(Math.min)
// --> true
```
