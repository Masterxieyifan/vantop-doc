## isLowerCase 方法
::: tip 常规用法
isLowerCase(value): boolean 

若value值为小写字母，则返回true，否则返回false。
:::


### isLowerCase 参数列表
| 参数          | 说明                 | 类型     | 默认值   |
| :----------- |:-----------------| :--------| :--------|
| value         | 需要验证是否为小写字母的值     | string       | -      |

#### 例子

```js
isLowerCase('lower case')
// --> true

isLowerCase('Lower Case')
// --> false
```
