## isEmail 方法
::: tip 常规用法
isEmail(value): boolean 

若value值为正确的邮箱地址，则返回true，否则返回false。
:::


### isEmail 参数列表
| 参数          | 说明                 | 类型     | 默认值   |
| :----------- |:-----------------| :--------| :--------|
| value         | 需要验证是否为邮箱地址的值     | string       | -      |

#### 例子

```js
isEmail('yuancheng.cai@vantopgroup.com')
// --> true
```
